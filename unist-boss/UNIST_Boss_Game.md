# UNIST Boss Game

## 컨셉
자쿰 오마주 보스전. 교수 캐릭터(패러디, 실명 X) 8개 팔에 학점 관련 무기.
클리어 = A+ 엔딩, 실패 = 재수강 엔딩.

## 스택
- Python + Pygame
- 사용 모델: Claude Sonnet
- 2D, 고퀄 지향 (레바의 모험 감성)

## 스코프
챕터 1개, 보스 1마리, 팔 패턴 8개, 인트로 컷씬 + 승패 엔딩

## 구조
/assets (스프라이트, 사운드)
/scenes
  intro.py
  boss.py
  ending.py
main.py

## 팔 목록 (자쿰 오마주)
1. F 낫 — 즉사기
2. 재수강 사슬 — 강제 리셋
3. D+ 창 — 근접 예측형
4. 출석 도장 망치 — 광역 쿵
5. 레퍼런스 부족 채찍 — 원거리 훅
6. 결석 사유서 방패팔 — 본체 무적 담당, 마지막에 부숴야 함
7. 팀플 무임승차 갈고리 — 플레이어 끌어당김
8. 디펜스 압박 부메랑 — 각도 계산형 투사체

## 퀄리티 기준
- 프레임 단위 스프라이트 애니메이션 (idle/walk/attack/hit)
- 파티클 시스템 (타격 이펙트, 팔 파괴 이펙트)
- 화면 흔들림 (screen shake on hit/광역기)
- 픽셀 폰트 적용 (UI/데미지 텍스트)
- 사운드 이펙트 + BGM

## 개발 순서
1. 게임 루프 + 플레이어 이동/공격 (placeholder 사각형)
2. 팔 1개(F 낫) 붙여서 충돌판정 + 화면흔들림/파티클 테스트
3. 나머지 팔 로직 붙이기
4. 스프라이트/사운드 입히기 (제일 마지막)
